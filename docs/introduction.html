Introduction #

What is it ? #

A μJam template for web based project or product documentation. Different markdown documentation files are accessible via a common sidebar.

Where is it from ? #

It is one of several basic templates for μJam - the minimalistic Jamstack approach for building GitHub-Pages hosted websites.

Who can use it ? #

This template is open source and can be used by by everyone.
